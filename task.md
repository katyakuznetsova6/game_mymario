# 🎮 Проект: 2D платформер (в стиле Super Mario)

## Цель
Создать простую 2D игру-платформер, работающую в браузере, с базовой механикой Mario: движение, прыжок, платформы, враг.

---

## Технологии
- HTML (один файл index.html)
- JavaScript
- Phaser.js (через CDN, без установки)

---

## Требования к структуре
- Весь код в одном файле index.html
- Внутри:
  - подключение Phaser через CDN
  - тег <script> с логикой игры
- Код должен запускаться сразу при открытии файла в браузере

---

## Игровая механика

### Игрок
- форма: квадрат
- цвет: оранжевый (#FF7A1A)
- управление:
  - стрелки ← → — движение
  - пробел — прыжок
- ограничение: не выходит за границы экрана

---

### Физика
- гравитация включена
- игрок падает вниз
- прыжок работает только при касании земли

---

### Платформы
- одна большая платформа (земля)
- минимум 2 платформы в воздухе
- игрок может на них запрыгивать

---

### Враг
- один враг (квадрат другого цвета)
- двигается автоматически влево-вправо
- отскакивает от границ
- при столкновении с игроком:
  - перезапуск сцены

---

## Визуал

### Стиль
- тёмный фон (#0F0F0F)
- минимализм
- без картинок, только фигуры

### Цвета
- игрок: оранжевый
- враг: красный
- платформы: тёмно-серые

---

## Камера
- фиксированная
- без прокрутки

---

## Управление
- клавиатура (стрелки + пробел)

---

## Требования к коду
- код должен быть рабочим без ошибок
- не использовать внешние файлы
- не использовать сложную архитектуру
- не использовать загрузку ассетов

---

## Результат
После запуска должен быть:
- виден игрок
- видны платформы
- работает движение и прыжок
- есть враг
- есть перезапуск при столкновении
